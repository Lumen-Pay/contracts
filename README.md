# LumenPay

**Stripe for Stellar — Non-custodial B2B crypto payment infrastructure**

---

## Problem

Global B2B payments are:

* Slow (1–5 days settlement)
* Expensive (2–5% in fees)
* Fragmented across banks, currencies, and intermediaries
* Hard to integrate for developers building modern fintech products

Crypto exists, but:

* Too complex for businesses
* Poor developer tooling
* Custody risks and compliance challenges

---

## Solution

**LumenPay** is a non-custodial payment processing layer built on the Stellar network.

We provide:

* Simple APIs for accepting and sending payments
* Stablecoin-first settlement (USDC, tokenized fiat)
* Instant, low-cost transactions
* Built-in compliance and identity flows

**We don’t hold funds — we orchestrate payments.**
